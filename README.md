# Knowledge Base

This is the knowledge base for KB Service.

## Structure

```
.
├── hardware/       # Hardware troubleshooting and guides
├── software/       # Software documentation
├── faq/           # Frequently asked questions
└── docs/          # General documentation
```

## Supported Formats

- `.md` - Markdown files
- `.txt` - Plain text files
- `.pdf` - PDF documents
- `.docx` - Microsoft Word documents

## How to Add Knowledge

1. Create a new file in the appropriate folder
2. Use clear titles and structure
3. Add tags for better searchability
4. Commit and push to GitHub

KB Service will automatically sync and index new content.

## File Naming

- Use lowercase with hyphens: `sensor-troubleshooting.md`
- Be descriptive: `installation-guide.md` not `guide.md`
