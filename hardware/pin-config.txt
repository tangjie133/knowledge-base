=====================================================
                SENSOR PIN CONFIGURATION
=====================================================

MODEL: DFRobot_HumanDetection
VERSION: 1.0

PINOUT:
--------
VCC  -> 5V
GND  -> GND
TX   -> RX (Pin 0)
RX   -> TX (Pin 1)

BAUD RATE: 115200

NOTES:
- Use hardware serial if possible
- Software serial may lose data at high baud
- Ensure stable 5V power supply

TROUBLESHOOTING:
----------------
No data:
1. Check baud rate matches
2. Swap TX/RX if needed
3. Verify 5V on VCC pin

Garbage data:
1. Check ground connection
2. Reduce baud rate to 9600
3. Use shorter wires
